# Music-Review-Front-End

**Description:**
This is a simple music review app built in React with a Rails back-end. Users can write reviews, comment on reviews, and follow reviewers.
The insperation for this app came from personal interest and was mostly built for fun.

**Technologies Used:**
**Front-End:**
- React
- react-router-dom
- Javascript
- HTML
- CSS

**Back-End:**

- Ruby On Rails
- Sessions
- Spotify API
- Ruby


**Setup:**

First you need to install and run the back-end you can find those intructions [here](https://www.example.com).
Then you can install and run the fron-end.

```
git clone https://github.com/Gabe-Corral/Music-Review-Front-End.git
cd Music-Review-Front-End
npm install
npm start
```
